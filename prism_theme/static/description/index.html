PRISM Theme

A modern, clean theme for the PRISM ERP system

Overview

The PRISM Theme module completely transforms the look and feel of Odoo to match the PRISM branding. It provides a fresh, modern interface that enhances user experience while maintaining all the powerful functionality of the underlying system.

Custom Branding

Replaces all Odoo branding with PRISM branding, including logos, favicons, and text references.

Modern UI

Implements a clean, modern user interface with improved typography, colors, and spacing.

Responsive Design

Fully responsive design that works beautifully on desktops, tablets, and mobile devices.

Custom Login Page

Redesigned login page with PRISM branding and a more user-friendly layout.

Enhanced Navigation

Improved navigation elements for better usability and faster access to key features.

Consistent Experience

Provides a consistent visual experience across all modules and pages.

Key Features

  • Complete rebranding of all Odoo elements to PRISM
  • Custom color scheme and typography
  • Redesigned login page
  • Enhanced form, list, and kanban views
  • Improved buttons and interactive elements
  • Custom icons and visual elements
  • Responsive design for all screen sizes

Installation

Install this module to apply the PRISM theme to your Odoo installation. No additional configuration is required. The theme will be applied automatically to all pages and views.
