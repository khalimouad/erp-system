PRISM Product

Moroccan-compliant product management for PRISM ERP

Overview

The PRISM Product module extends the standard product models with Moroccan-specific fields and functionality. It provides a comprehensive set of features to manage products and services in compliance with Moroccan regulations.

VAT Rate Management

Manage products with the correct VAT rates according to Moroccan tax law:

  • Standard Rate (20%)
  • Reduced Rate (14%)
  • Reduced Rate (10%)
  • Reduced Rate (7%)
  • Exempt (0%)

With automatic validation for service-specific VAT rates.

Product Classification

Categorize products based on their type:

  • Raw Materials
  • Semi-Finished Products
  • Finished Products
  • Consumables
  • Services

Inventory Management

Enhanced inventory control features:

  • Minimum Stock Levels
  • Maximum Stock Levels
  • Reorder Quantities
  • Lead Time Tracking
  • Shelf Life Management

Accounting Integration

Seamless integration with accounting:

  • Purchase Accounts
  • Sale Accounts
  • Stock Accounts
  • Default Accounts by Category

Units of Measure

Enhanced UoM management:

  • Local Names (Arabic)
  • Abbreviations
  • Detailed Descriptions

Multiple Views

Access products through different views:

  • Kanban View with Classification and VAT Rate
  • List View with Customizable Columns
  • Detailed Form View with Moroccan-specific Fields

Key Features

  • Comprehensive product management with Moroccan-specific fields
  • VAT rate management according to Moroccan tax law
  • Enhanced inventory control features
  • Accounting integration
  • Advanced search and filtering capabilities
  • Product categorization with inheritance of properties

Installation

Install this module to enable Moroccan-specific product management features in your PRISM ERP system. This module depends on the base, product, uom, and account modules.
